# Security Policy

## Supported Versions

Only the latest release/commit on the `main` branch is supported with security updates.

| Version | Supported |
| ------- | --------- |
| Latest  | :white_check_mark: |
| < Latest | :x: |

---

## Reporting a Vulnerability

If you discover a security vulnerability in Spydeck, please report it to us as soon as possible.

* **Email:** Send your vulnerability report directly to [contact@spydeck.dev](mailto:contact@spydeck.dev).
* **Details:** Please include a detailed description of the vulnerability, steps to reproduce it, and any potential impact or proof of concept.

We appreciate your help in keeping Spydeck secure and will do our best to acknowledge and address reports promptly.
